## ▶️ How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/your-repository.git
   ```

2. Open the project in your preferred C++ IDE or editor.

3. Compile the program:
   ```bash
   g++ filename.cpp -o output
   ```

4. Run the executable:
   ```bash
   ./output
   ```
